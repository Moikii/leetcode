# LeetCode Solutions

This repository contains my solutions to various LeetCode problems.

Learing about new data-structures and algorithms, improving overall problem-solving skills, but also showing my thinking and knowledge are the goals of these challanges for me.

## Content

Each problem has this own directory with the following files:
- `README.md`
    - Link to the LeetCode problem
    - Short introduction to the challange
    - Notes on my approach
- `solution.py`
    - Implementation of the solution
